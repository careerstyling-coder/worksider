## 1. 핵심 인사이트

사용자가 Workside에서 느껴야 할 것:
- **랜딩**: "여기 사람들이 움직이고 있구나"
- **진단 후**: "나만 하는 게 아니구나"
- **피드에서**: "여기 살아있네"

→ 모든 기능이 **"나 혼자가 아니라 함께하고 있다"**는 느낌을 줘야 함

## 3. 서비스 소개 페이지 콘텐츠 방향

### 핵심 메시지 구조

```
1. 공감 (Hook)
   "나는 일을 어떻게 하는 사람일까?"
   "왜 같은 회의에서 어떤 사람은 답답하고, 어떤 사람은 신나는 걸까?"

2. 발견 (Solution)
   "3분이면 나의 Workstyle DNA를 발견할 수 있어요"
   4축 설명: 실행력 / 협력 / 영향력 / 자율성

3. 연결 (Community)
   "나와 비슷한 사람들이 여기 있어요"
   "함께 궁금해하고, 함께 답을 찾아가요"

4. 성장 (Value)
   "매주 새로운 질문, 매번 새로운 발견"
   "진단 → 질문 → 인사이트 → 성장의 선순환"
```

### 소개 페이지 섹션 (안)

1. **히어로**: 공감 질문 + 진단 CTA
2. **DNA 4축 소개**: 각 축이 뭘 의미하는지 시각적으로
3. **이런 분들이 쓰고 있어요**: 실시간 참여자 수 + 유형 분포
4. **함께 성장하는 방법**: 진단 → 공유 → 질문 → 성장 플로우
5. **사용자 후기**: 실제 반응
6. **시작하기 CTA**
