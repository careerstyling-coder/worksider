## Phase 4: 디오티마 사다리

### Layer 0: 구체적 사례
MBTI (16Personalities): 무료 성격 진단으로 월 2,000만+ 방문, 4글자 라벨 SNS 공유 문화. 그러나 일 특화가 아닌 성격 전반.

### Layer 1: 패턴
- 패턴 A: "나는 어떤 사람인가?" 자기 정의 욕구는 보편적. 라벨이 붙으면 공유한다.
- 패턴 B: 기존 일 진단(DISC, StrengthsFinder)은 B2B 중심이라 개인 접근성 낮고 바이럴 안 된다.
- 패턴 C: 바이럴 되는 진단(MBTI, Wrapped)은 일 특화가 아니다. **일 특화 + 바이럴 = 빈 자리.**

### Layer 2: 규칙
**규칙**: 사람들은 정체성을 객관화하는 도구를 사용하고 공유한다. 단, (1) 무료 (2) 즉시 결과 (3) 공유 가능한 라벨일 때. 특정 맥락에 특화될수록 더 깊은 가치.
**적용 범위**: 자기 이해 욕구가 있는 모든 도메인
**한계**: 결과가 너무 복잡하면 공유 안 됨 / 즉각적 "아하!" 순간 필수
**반례**: StrengthsFinder — 좋은 도구이나 유료($20+)라서 바이럴 실패 → "무료" 조건 확인

### Layer 3: 도메인 전이
| 전이 대상 | 적용 가능? | 근거 |
|----------|-----------|------|
| 건강/피트니스 (Apple Watch) | 예 | 무료 측정 + 즉시 결과 + "만보" 공유 |
| 금융 (토스 소비 리포트) | 예 | 무료 + 즉시 시각화 + 소비 패턴 공유 |
| 교육 (듀오링고) | 예 | 무료 + 레벨 + "365일 스트릭" 공유 |
| 게임 (LOL 티어) | 예 | 무료 + 랭크 + "다이아몬드" 라벨 공유 |

### Layer 4: 메타 학습
**핵심 돌파구**: 기존 서비스를 "바이럴 축"과 "일 특화 축"의 2x2로 놓은 순간 빈 자리가 보임
**메타 패턴**: "기존 성공 사례를 2개 축으로 매핑하면, 두 축 모두 높은 사분면이 비어있을 때 그곳이 기회다"
**재사용 가능한 질문**: (1) 성공한 것들의 공통 조건은? (2) 그 조건 충족하면서 빈 맥락은? (3) 왜 아직 아무도 없는가?

### Layer 5: 이데아
**불변 원리**: "사람은 자기 자신을 이해하고 싶어하고, 그 이해를 타인과 나누고 싶어한다. 이 욕구가 특정 맥락에 구체화될 때 가치가 생기고, 공유 가능한 언어가 만들어질 때 커뮤니티가 형성된다."
